1. Aceitação: Ao utilizar o Simbiose, você concorda com estes Termos de Uso e Política de Privacidade.

2. Papéis e Responsabilidades (LGPD): O Simbiose atua como uma ferramenta de gestão. O Administrador do Sistema (Condomínio/Usuário) é o Controlador dos dados pessoais tratados, sendo responsável pela conformidade com a LGPD perante os titulares (moradores). O Simbiose fornece os meios técnicos para o tratamento.

3. Armazenamento "Local-First": O Simbiose adota a filosofia "Local-First". Seus dados residem primariamente no dispositivo local. O tratamento de dados pessoais ocorre, em regra, no ambiente controlado pelo Controlador.

4. IA e Privacidade: O processamento de imagens para leitura de etiquetas é realizado de forma transitória (em memória). Imagens de rostos humanos são bloqueadas ou ignoradas por padrão. Nenhuma imagem é armazenada permanentemente pelo sistema, salvo se explicitamente configurado pelo Administrador em local de sua responsabilidade.

5. Backups e Segurança: A responsabilidade pela integridade, segurança e conformidade dos backups (seja no Cofre Local ou em serviços de Nuvem de terceiros) é inteiramente do Administrador do sistema. Recomendamos fortemente o uso de criptografia em todos os destinos de backup.

6. Direitos dos Titulares: O Administrador compromete-se a atender às solicitações dos titulares de dados (moradores) conforme previsto na LGPD, incluindo acesso, correção, anonimização ou exclusão de dados pessoais tratados pelo sistema.

7. Licença: O uso deste software está condicionado a uma licença ativa (exceto no modo Guest limitado). A ausência de licença pode restringir o acesso a funcionalidades de processamento.
